# React

## How to create an app
- ```nvm list``` ***to list installed node version***
- ```nvm use v22.1.0``` ***use/check node version of your choice for your project***
- ```npm create vite@latest ui_human_serveillance``` ***create react app named ui_human_serveillance***
- ``npm install``
- ``npm run dev``
